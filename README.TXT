----------------- INSTALLATION ---------------------

Bonjour

Voici le lien GitHub pour cloner le projet : 
https://github.com/StephCodeTraining/ecf_studi_garage.git

Après avoir sélectionner votre dossier dans lequel vous aurez 
créé un dépôt git avec l'invite de commande
(git init)

Il vous suffira de cloner le projet à l'interieur.
(git clone url)

Cette évaluation a été éfféctuée en locale sous WampServeur.

Le site nécessite de démarrer un serveur local afin d'ouvrir
la page de démarage en utilisant le "localhost".

Le fichier de démarage se trouve dans le dossier installation
et se nomme "init_site.php"

Pour ma part le chemin complet vers le projet est : 
http://localhost/training/ECF/site_garage_parrot/installation/init_site.php

(le dossier {training} n'est pas à prendre en compte pour vous 
car mon depôt git a été éffectuer dans le dossier {ECF})

Une fois ouvert il vous faut entrer vos données d'administrateur
afin de pouvoir vous connecter sur le site.

Vous pouvez vous référer à la charte graphique dans le dossier "Annexes"
pour avoir un aperçu des différentes fonctionalitées.

Le dossier complet est égalemment transmis via la plate-forme.
